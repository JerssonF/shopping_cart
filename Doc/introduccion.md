# Introducción sencilla al proyecto

Este proyecto es una guía práctica para entender y crear un carrito de compras. Está pensada para quien quiera ver, paso a paso, cómo montar una pequeña aplicación que permita guardar productos mientras se decide comprarlos.

¿Qué busca este proyecto?

- Enseñar de forma clara cómo funciona un carrito de compras en una aplicación.
- Mostrar, con ejemplos, cómo organizar las partes básicas de la aplicación (la parte que recibe las solicitudes, la parte que guarda los datos y la página que ve el usuario).
- Servir como ejemplo para aprender y adaptar en proyectos personales.

Qué hace la aplicación (explicado de forma simple)

- Permite agregar productos a un carrito.
- Permite ver qué hay en el carrito y cuántos productos hay.
- Permite cambiar la cantidad de un producto o quitarlo.
- Muestra el total a pagar por los productos seleccionados.
- Guarda la información para que se conserve entre sesiones.

Alcance y organización

El proyecto incluye ejemplos de:

- La parte que procesa las solicitudes y aplica las reglas (lógica de negocio).
- La página web donde el usuario ve y gestiona su carrito.
- La forma de guardar la información (archivos o bases de datos).
- Un modo de ejecutar todo junto para probarlo fácilmente.

Cómo usar esta guía

Se explica el proyecto en etapas: primero lo básico para que funcione, luego mejoras opcionales. Cada paso está descrito para que puedas seguirlo y probarlo por tu cuenta.

Nota final

Esta guía facilita aprender practicando. No es necesario tener experiencia profunda: se usan ejemplos claros para entender cada parte.

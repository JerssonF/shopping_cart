# HU-005: Quitar un producto del carrito

Como usuario quiero sacar un producto de mi carrito cuando ya no lo quiero comprar.

Criterios de aceptación

- Puedo eliminar un producto del carrito.
- El producto deja de aparecer en la lista del carrito.
- El total del carrito se actualiza después de quitar el producto.
- Si intento eliminar algo que no está, recibo un mensaje claro.

Notas sencillas

- La acción debe ser reversible antes de confirmar la compra (por ejemplo, puedo añadirlo de nuevo).

Etiquetas: eliminar, carrito, producto

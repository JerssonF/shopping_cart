# HU-006: Mostrar el total del carrito

Como usuario quiero ver el total de mi carrito para saber cuánto debo pagar.

Criterios de aceptación

- El total muestra la suma de todos los subtotales de los productos.
- El total cambia cuando agrego, quito o cambio la cantidad de productos.
- Se muestra el número total de productos y el monto final.

Notas sencillas

- El total ayuda a decidir si sigo comprando o finalizo la compra.

Etiquetas: total, carrito, cálculo

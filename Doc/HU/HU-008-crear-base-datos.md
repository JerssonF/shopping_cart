# HU-008: Preparar la base de datos

Como equipo queremos tener la estructura inicial de la base de datos para guardar carritos y productos.

Criterios de aceptación

- Existen los scripts para crear las tablas principales (carrito y productos del carrito).
- La estructura permite relacionar un carrito con sus productos.
- Los archivos están listos para conectar con el código del proyecto.

Notas sencillas

- No es necesario detallar rutas personales; los archivos deben quedar dentro del repositorio del proyecto.
- Esta historia cubre sólo la preparación de la base de datos, no su despliegue final.

Etiquetas: database, esquema, persistencia

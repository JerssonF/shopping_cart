# HU-001: Crear un carrito

Como usuario quiero que el sistema me cree un carrito vacío para poder guardar productos mientras decido comprarlos.

Criterios de aceptación

- Se crea un carrito nuevo para el usuario cuando no tiene uno activo.
- El sistema devuelve la información básica del carrito (por ejemplo, su identificador).
- Si ya hay un carrito abierto para el usuario, no se crea otro; se devuelve el existente.

Notas (lenguaje sencillo)

- El carrito guarda los productos que quiero comprar y permanece hasta que lo cierre o confirme la compra.
- Esta historia ya está implementada en el proyecto.

Etiquetas: carrito, usuario, funcionalidad

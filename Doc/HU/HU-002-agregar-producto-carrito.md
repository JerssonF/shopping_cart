# HU-002: Añadir un producto al carrito

Como usuario quiero poder poner un producto en mi carrito para tenerlo guardado hasta comprar.

Criterios de aceptación

- Puedo añadir un producto indicando la cantidad.
- Si el mismo producto ya está en el carrito, se aumenta su cantidad en lugar de duplicarlo.
- Se ve el subtotal del producto (precio × cantidad).
- La lista del carrito se actualiza para que yo la vea.

Notas sencillas

- El carrito mantiene nombre, cantidad y precio de cada producto para mostrar lo que elegí.
- Esta historia describe el comportamiento visible para el usuario, sin tecnicismos.

Etiquetas: añadir, carrito, producto

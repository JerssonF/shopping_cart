# HU-003: Ver mi carrito

Como usuario quiero ver lo que hay en mi carrito para saber qué productos seleccioné y cuánto costarán.

Criterios de aceptación

- Puedo ver la lista de productos del carrito.
- Para cada producto aparece la cantidad y el precio por unidad.
- Puedo ver el subtotal por producto y el total general del carrito.
- Si no existe el carrito, se muestra un aviso claro.

Notas sencillas

- Esta vista me permite revisar antes de pagar.
- La información mostrada debe coincidir con lo que guardé en el carrito.

Etiquetas: ver, carrito, consulta

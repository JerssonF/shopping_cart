# HU-004: Cambiar la cantidad de un producto

Como usuario quiero poder cambiar cuántas unidades de un producto hay en mi carrito para ajustarlo a lo que realmente quiero comprar.

Criterios de aceptación

- Puedo modificar la cantidad de un producto ya agregado.
- La cantidad nueva debe ser mayor que cero.
- Al cambiar la cantidad se actualiza el subtotal del producto y el total del carrito.
- El cambio se refleja cuando vuelvo a ver el carrito.

Notas sencillas

- Si intento poner cero o un valor inválido, el sistema debe avisarme.
- Esta función ya está implementada en el proyecto.

Etiquetas: actualizar, cantidad, carrito

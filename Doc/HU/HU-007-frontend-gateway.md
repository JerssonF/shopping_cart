# HU-007: Usar el carrito desde la web

Como usuario quiero usar una página web para ver y cambiar mi carrito, de forma simple y cómoda.

Criterios de aceptación

- Puedo ver los productos del carrito desde la interfaz.
- Puedo añadir, cambiar cantidad y quitar productos desde la interfaz.
- Las acciones realizadas en la web se reflejan en el carrito que guarda el sistema.

Notas sencillas

- La interfaz actúa como puente para que mis acciones lleguen al sistema y se guarden.
- Esta historia describe lo que el usuario ve y hace, sin entrar en detalles técnicos.

Etiquetas: frontend, interfaz, usuario
